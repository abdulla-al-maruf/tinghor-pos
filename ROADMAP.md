# Tinghor POS Roadmap

This roadmap shows the planned development direction for Tinghor POS.

## Version 0.1.0 — Public Preview

* Clean public repository structure
* Add README, license, screenshots, and setup guide
* Add live demo link
* Verify environment variable safety
* Confirm core POS, inventory, ledger, and reporting modules
* Prepare first public release

## Version 0.2.0 — Stability and Testing

* Improve error handling across core modules
* Add more form validation
* Improve loading and empty states
* Add basic unit tests for important utilities
* Improve database policy documentation
* Fix UI issues reported by early testers

## Version 0.3.0 — Business Workflow Improvements

* Improve receipt format
* Add better customer due/payment tracking
* Add purchase return and sales return support
* Improve low-stock alerts
* Add export options for sales and ledger data
* Improve Bengali translations

## Version 0.4.0 — Security and Admin Controls

* Improve role-based access control
* Add better audit logs
* Review Supabase RLS policies
* Add admin activity reports
* Improve user permission documentation

## Version 1.0.0 — Stable Release

* Stable POS workflow
* Stable inventory and ledger system
* Complete documentation
* Tested deployment guide
* Production-ready database setup guide
* Public demo and example dataset
