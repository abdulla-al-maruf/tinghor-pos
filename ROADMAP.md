# Tinghor POS Roadmap

Tinghor POS is a Bangladesh-first POS, inventory, and ledger management system for local corrugated sheet / Dhew Tin businesses.

This roadmap explains the planned direction of the project and helps contributors, testers, and users understand what is already available, what is being improved, and what is planned next.

## Project Goals

* Build a simple and practical POS system for local corrugated sheet businesses
* Support Bengali-first workflows for Bangladeshi small businesses
* Make sales, inventory, customer dues, purchases, expenses, and reports easier to manage
* Provide an open-source foundation that can be adapted for similar local business types
* Improve security, stability, and documentation for real-world use
